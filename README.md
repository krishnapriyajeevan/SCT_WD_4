## Premium To-Do Web Application (Task 4)
This is a feature-rich and interactive To-Do Web Application built as part of my Web Development internship at SkillCraft Technology.

### 🚀 Features:
- Add, edit, and delete tasks smoothly.
- Task categorization (e.g., Pending, Completed) or priority tags.
- Local Storage integration to save tasks even after refreshing the page.
- Beautiful, clean, and minimalist user interface.

### 🛠️ Technologies Used:
- HTML5
- CSS3 (Modern layouts and interactive active states)
- JavaScript (Local Storage, Array methods, and DOM actions)
